SCBC Type Ramp

The SCBC Type Ramp is based on a modular scale of 1.5. The base font size is 18px, with each size above it multiplied by 1.5 and each size below it divided by 1.5. There are 7 total sizes that are categorized according to the following:

Headings
There are 5 headings to correspond to the semantic HTML tags. h1 and h2 have corresponding subheadings which are detailed in the subheading section below.
  • h1 & .type-h1
  • h2 & .type-h2
  • h3 & .type-h3
  • h4 & .type-h4
  • h5 & .type-h5
Subheadings
These are to be used in conjunction with their appropriate headings. For instance, when using .type-h1 you would use .type-sh1.
  • .type-sh1
  • .type-sh2
Paragraphs, captions, ect...
The remaining sizes are used for paragraphs & captions.
  • p & .type-t5
  • .caption & .type-t6
  • .caption-small & .type-t7

SCBC Color Palette

The SCBC Color Pallette consists of 7 base colors, each with 10 tinted variations. Syntax for variables and classes follow an object oriented pattern. For example: $c-white where "c" denotes color followed by the name.

If you're using the source SASS files:

You'll have access to variables for each base color and all of it's tints. For example: $c-white-, $c-white-10, etc...

If you're using the CSS file:

You'll have access to classes for each base color. There are background, fill and border classes for each tint. For example: .c-bg-white which renders background: #fff, .c-fill-white which renders color: #fff, and .c-line-white which renders border-color: #fff.
